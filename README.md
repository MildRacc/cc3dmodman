# cc3dmodman (Crazy Cattle 3D Mod Manager)

Simple mod manager/mod loader for Crazy Cattle 3D.

![gameplay](docs/media/gameplay.gif)

## Use
1. Extract the folder anywhere.
2. Run `cc3dmodman`.
3. Select the Crazy Cattle 3D executable when asked.
4. Launch.

## Included mods
- Gun mod
- Rocket booster mod
- Hat + glasses mod
- In-game overlay adjustment mod

## Notes
- Mods are temporary and stay inside this folder. The manager uses a small local copy and doesn’t permanently change your game install.
- To uninstall, delete the folder.
- `godotpcktool` is required. Don’t move/delete/touch it.

## Writing your own mods
There’s full documentation in `docs/` (modding API + how-to) for making your own mods.
